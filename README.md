# Dashboard de Gestion de Stocks

## Description
Ce projet est le résultat final de ma formation en **HTML et CSS**.  
Il illustre la mise en pratique de l’ensemble des compétences acquises durant le parcours : structure sémantique, mise en page moderne, utilisation des grilles CSS, gestion du responsive design et application d’un style-guide complet.

Le dashboard permet de gérer les **stocks de produits électroniques**, avec une interface claire, professionnelle et responsive.  
Il comprend une barre latérale de navigation, des cartes de statistiques, un tableau de gestion des stocks et un système d’actions interactives.

## Objectifs du Projet
- Parfaire et finaliser la formation en **HTML/CSS**
- Mettre en pratique tous les acquis sur un projet complet
- Concevoir une interface moderne, claire et responsive
- Appliquer un guide de style professionnel

## Fonctionnalités Principales
- Tableau de bord principal avec statistiques (produits, stocks, valeur totale…)
- Système de notifications avec badge animé
- Tableau de gestion des produits (ID, catégorie, stock, prix, statut, actions)
- Boutons d’action interactifs (ajouter, exporter, modifier, supprimer)
- Palette cohérente et hiérarchie typographique respectée

## Technologies Utilisées
- HTML5
- CSS3 (Flexbox + Grid Layout)
- Font Awesome  (pour les icônes)
